# Aplicatie-Calendar-C-
Scopul aplicației este de a ajuta utilizatorii să își organizeze sarcinile zilnice într-un mod eficient. Fiecare sarcina poate fi adăugată cu un nume, prioritate și dată limită, iar utilizatorii pot marca sarcinile finalizate și le pot sorta după prioritate. Acest proiect a fost facut ca proiect pentru interviul de selectie in proiect Erasmus.
